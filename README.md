# School-Staff-And-Parent-App


School management mobile app built with React Native CLI and FastAPI for staff and parents. Features include attendance tracking, lesson planning, student analytics, fee management, event updates, JWT authentication, MongoDB integration, offline sync, and direct parent-teacher communication for improved school operations.

- School Staff app streamlines administrative tasks for school staff, including attendance tracking, lesson planning, and student performance
analytics, to boost operational efficiency in educational institutions.

- It tackles inefficiencies in traditional school management systems, such as manual record-keeping and delayed communication, which often lead to
errors during peak academic seasons like exams.

- Developed using React Native CLI with a Node.js backend, it features JWT-based authentication for secure access and MongoDB for robust data
storage, with offline sync capabilities to support fieldwork in remote areas; the app is scheduled for Play Store publication in Q4 2025.

- Designed for school parents, this app provides intuitive tools for monitoring child development, event calendars, fee payments, and direct
messaging with educators to foster stronger school-home partnerships.

- The solution mitigates communication gaps between parents and schools, a persistent issue exacerbated by hybrid learning models post-pandemic,
ensuring parents stay informed without overwhelming emails or calls.
